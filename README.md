# Weather_App
## Created for Palmetto

**Dev steps:** 

* npx create-react-app
* add react-test-renderer, jest-dom

**Run npm i** 

**Run npm start**

## To Build For Prod Run:
**npm build**

## Available Scripts
* install
* start
* test
* build

Enjoy! 
**please note** You need to create an env file in src with:
```REACT_APP_API_KEY=your API Key here```
